
# Variáveis Compostas — Listas
> O repositório tem como objetivo apresentar minha jornada de aprendizado com exercícios práticos sobre variáveis compostas, com foco em **listas** no Python, adicionados conforme vou resolvendo e organizados por nível de dificuldade.

---

## ✦ Níveis de Dificuldade

| Nível | Descrição |
|---|---|
| 🟢 **Iniciante** | Conceitos básicos, lógica simples |
| 🟡 **Intermediário** | Combinação de conceitos, lógica mais elaborada |
| 🔴 **Avançado** | Problemas mais complexos, múltiplos conceitos |

---
